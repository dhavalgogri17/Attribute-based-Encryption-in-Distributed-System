Software Requirement

.Net FrameWork 3.5 or more

Hardware Requirement

Steps
1.The user has to first register to use this software.
2.After registering he can login.
3.The user can upload a text file which is encrypted with scrambled attributes.
4.The user can request the server for secret key for querying or retieval of data.
5.The user can download files from server that match his attributes accordingly.
6.The user can only decrypt the files if authenticated to do so.

Input : Text Files

Output : Encrypted File with Scrambled Attributes
